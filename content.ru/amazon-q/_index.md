# Amazon Q Developer

Amazon Q Developer — это помощник разработчика на основе ИИ, запущенный AWS, который предоставляет разработчикам возможности интеллектуальной генерации кода, отладки и оптимизации.

Эта документация дополняет официальный FAQ, и информация может быть неактуальной. Пожалуйста, в первую очередь проверяйте [официальный FAQ на сайте](https://aws.amazon.com/q/developer/faqs/) для вопросов.
{{% /hint %}}

## Kiro и Amazon Q Developer

Kiro и Amazon Q Developer — это помощники разработчика на основе ИИ, запущенные AWS. Если у вас есть подписка на Amazon Q Developer, вы можете использовать как Kiro, так и Amazon Q Developer одновременно.

- Kiro: Kiro — это **Agentic IDE**, построенная на Visual Studio Code с расширенными возможностями ИИ.
- Amazon Q Developer: В основном включает два продукта: **Amazon Q Developer IDE Plugin** и **Amazon Q Developer CLI**.
