# Amazon Q Developer

Amazon Q Developer — это ассистент разработки на основе ИИ, запущенный AWS, который предоставляет разработчикам интеллектуальные возможности генерации кода, отладки и оптимизации.

{{% hint warning %}}
Эта документация дополняет официальные FAQ, и информация может быть неактуальной. Пожалуйста, в первую очередь проверяйте [FAQ официального сайта](https://aws.amazon.com/q/developer/faqs/).
{{% /hint %}}

## Kiro и Amazon Q Developer

И Kiro, и Amazon Q Developer — это ассистенты разработки на основе ИИ, запущенные AWS. Если у вас есть подписка на Amazon Q Developer, вы можете использовать и Kiro, и Amazon Q Developer одновременно.

- Kiro: Kiro — это **Agentic IDE**, построенная на Visual Studio Code с улучшенным опытом работы с ИИ.
- Amazon Q Developer: В основном включает два продукта: **Amazon Q Developer IDE Plugin** и **Amazon Q Developer CLI**.
